# 코딩 연습 - 튜플 (2)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 기본 튜플 반환

`(int, int)` 튜플을 반환하는 
`Tally` 메서드를 작성하고, 반환된 튜플의 
`Item1`과 `Item2`를 출력하세요.

**실행 결과**

```
합계: 12, 개수: 3
```

---

## 2. 반환 튜플에 이름 지정

반환 타입에 `(int Sum, int Count)` 형태로 이름을 지정한 `Calculate` 메서드를 작성하고, 이름으로 결과에 접근하여 출력하세요.

**실행 결과**

```
합계: 45, 개수: 6
```

---

## 3. 표현식 본문으로 간결하게 작성

표현식 본문(`=>`)을 사용하여 
`(int first, int second)` 튜플을 반환하는 
`GetPair` 메서드를 작성하고 출력하세요.

**실행 결과**

```
100, 200
```

---

## 4. 최솟값과 최댓값 반환

정수 배열 `{ 5, 2, 8, 1, 9, 3 }`에서 
최솟값과 최댓값을 찾아 
`(int min, int max)` 
튜플로 반환하는 
`FindMinMax` 메서드를 작성하고 결과를 출력하세요.

**실행 결과**

```
최솟값: 1, 최댓값: 9
```

---

## 5. 기본값 반환

`default` 키워드를 사용하여 
`(int, int)` 튜플의 기본값(0, 0)을 반환하는 
`ZeroZero` 메서드를 작성하고 
결과를 출력하세요.

**실행 결과**

```
0, 0
```

---

## 6. 분해 기본

문자열과 정수를 포함하는 튜플을 생성하고, 
`(string name, int age) = bob;` 형태로 
분해하여 개별 변수로 추출한 후 출력하세요.

**실행 결과**

```
이름: Bob
나이: 23
```

---

## 7. var를 사용한 분해


`var (sum, count) = Tally();` 
형태로 튜플 반환값을 분해하고 출력하세요.

**실행 결과**

```
Sum: 12, Count: 3
```

---

## 8. 기존 변수에 분해

미리 선언된 변수에 튜플을 분해하여 
값을 할당하세요.

**실행 결과**

```
Alice, 30
```

---

## 9. 무시 패턴 (`_`)

3개 요소를 가진 튜플에서 
`_`(무시 패턴)를 사용하여 
나이만 추출하고 출력하세요.

**실행 결과**

```
나이: 23
```
