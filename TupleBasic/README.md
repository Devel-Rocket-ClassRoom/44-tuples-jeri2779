# TupleBasic

학생 정보를 튜플로 관리하는 프로그램을 작성하시오.

다음 기능을 구현해야 함:

1. **CreateStudent 메서드**: 이름, 나이, 학년을 받아서 튜플로 반환
   - 반환 타입: `(string name, int age, int grade)`

2. **PrintStudent 메서드**: 학생 튜플을 받아서 정보 출력
   - 형식: `"[이름] - 나이: [나이]세, 학년: [학년]학년"`

3. **Main**:
   - CreateStudent로 3명의 학생 생성
   - 각 학생 정보를 PrintStudent로 출력
   - 첫 번째 학생의 정보를 튜플 분해로 추출하여 개별 출력

## 예상 실행 결과

```
=== 학생 정보 출력 ===
철수 - 나이: 16세, 학년: 1학년
영희 - 나이: 17세, 학년: 2학년
민수 - 나이: 18세, 학년: 3학년

=== 첫 번째 학생 분해 ===
이름: 철수
나이: 16
학년: 1
```
