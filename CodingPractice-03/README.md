# 코딩 연습 - 튜플 (3)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. Equals 메서드

같은 값을 가진 두 튜플과 다른 값을 가진 튜플을 생성하고, `Equals` 메서드로 비교한 결과를 출력하세요.

**실행 결과**

```
True
False
```

---

## 2. == 및 != 연산자

`==`와 `!=` 연산자를 사용하여 두 튜플을 비교하고 결과를 출력하세요.

**실행 결과**

```
True
False
```

---

## 3. 타입 호환성 - 기본

요소 타입이 같은 두 튜플 간에 할당이 가능함을 확인하세요. 첫 번째 튜플을 이름이 다른 두 번째 튜플 변수에 할당하고 출력하세요.

**실행 결과**

```
Alice, 25
```

---

## 4. 타입 호환성 - 주의사항

요소 이름이 의미적으로 다르지만 타입이 맞으면 할당되는 경우를 확인하세요. `(string name, int age, char sex)` 튜플을 `(string age, int sex, char name)` 타입에 할당하고, 이름으로 접근 시 혼란스러운 결과가 나오는 것을 출력하세요.

**실행 결과**

```
Bob
23
M
```

---

## 5. 제네릭과 튜플 - 리스트

`List<(string name, int age)>` 타입의 리스트를 생성하고, 3명의 사람 정보를 추가한 후 `foreach`로 출력하세요.

**실행 결과**

```
Alice: 25세
Bob: 30세
Charlie: 35세
```

---

## 6. 딕셔너리의 키로 튜플 사용

`Dictionary<(int x, int y), string>` 타입의 딕셔너리를 생성하고, 좌표를 키로 사용하여 위치 이름을 저장한 후 출력하세요.

**실행 결과**

```
원점
오른쪽
```

---

## 7. System.Tuple 클래스

이전 방식인 `System.Tuple` 클래스와 `Tuple.Create` 메서드를 사용하여 튜플을 생성하고 `Item1`, `Item2`로 출력하세요.

**실행 결과**

```
Bob, 23
Alice, 25
```
