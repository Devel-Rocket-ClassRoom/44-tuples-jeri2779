# 튜플

## 사용 방법

1. Visual Studio에서 `Chapter44-Tuples.slnx`를 엽니다
2. 솔루션 탐색기에서 프로젝트를 선택하고 `README.md`를 읽습니다
3. `Program.cs`에 코드를 작성합니다
4. 해당 프로젝트를 시작 프로젝트로 설정하고 실행합니다 (`Ctrl+F5`)

## 코딩 연습

`CodingPractice` 프로젝트의 README를 따라 교안 예제 코드를 직접 작성하고 실행합니다.

| 프로젝트 | 내용 |
|----------|------|
| CodingPractice-01 | 튜플 기본, 요소 이름 지정 (8개 과제) |
| CodingPractice-02 | 메서드에서 튜플 반환, 튜플 분해 (9개 과제) |
| CodingPractice-03 | 튜플 비교, 타입 호환성, 제네릭과 튜플 (7개 과제) |

## 연습문제
