# 코딩 연습 - 튜플 (1)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 튜플 리터럴

괄호 안에 문자열과 
정수를 나열하여 튜플을 생성하고, 
`Item1`, `Item2`로 
각 요소에 접근하여 출력하세요.

**실행 결과**

```
Bob
23
```

---

## 2. 다양한 타입의 튜플

### 2-1. 같은 타입의 튜플

3개의 정수를 포함하는 튜플을 생성하고 `Item1`, `Item2`, `Item3`을 쉼표로 구분하여 출력하세요.

**실행 결과**

```
12, 34, 56
```

### 2-2. 서로 다른 타입의 튜플

문자열, 정수, bool, double을 포함하는 튜플을 생성하고 모든 요소를 쉼표로 구분하여 출력하세요.

**실행 결과**

```
Hello, 100, True, 3.14
```

---

## 3. 명시적 튜플 타입

### 3-1. 기본 명시적 타입

`(string, int)` 타입을 명시적으로 지정하여 튜플을 선언하고 출력하세요.

**실행 결과**

```
Alice, 25
```

### 3-2. 이름이 있는 명시적 타입

`(ushort Width, ushort Height)` 타입으로 해상도 튜플을 선언하고 "해상도: W x H" 형식으로 출력하세요.

**실행 결과**

```
해상도: 1920 x 1080
```

---

## 4. 튜플의 값 타입 특성

튜플을 다른 변수에 할당하여 복사본을 만들고, 복사본의 
`Item1`을 변경한 후 
원본과 복사본을 각각 출력하여
원본이 변경되지 않았음을 확인하세요.

**실행 결과**

```
원본: (Bob, 23)
복사본: (Joe, 23)
```

---

## 5. 생성 시 이름 지정

튜플 생성 시 `name`, `age`와 같이 요소 이름을 지정하고, 이름과 `Item1`, `Item2` 두 가지 방법으로 접근하여 출력하세요.

**실행 결과**

```
Bob
23
Bob
23
```

---

## 6. 요소 이름 지정 예제

`Name`, `IsStudent`, `OrderPrice` 이름을 가진 튜플을 생성하고, 문자열 보간을 사용하여 정보를 출력하세요. 금액은 천 단위 구분 기호를 포함하세요.

**실행 결과**

```
철수(학생: True) - 주문: 1,000원
```

---

## 7. 타입 선언에서 이름 지정

`(string name, int age, char gender)` 
형태로 타입을 선언하면서 이름을 지정하고, 
각 요소를 줄바꿈하여 출력하세요.

**실행 결과**

```
이름: Bob
나이: 23
성별: M
```

---

## 8. 요소 이름 자동 추론

`DateTime.Now`의 `Day`, `Month`, `Year` 
프로퍼티를 튜플에 넣으면 이름이 자동으로 추론됨을 확인하세요.

**실행 결과**

```
일: 27
월: 12
년: 2025
```
